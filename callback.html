Callback Function

  • A callback is a function that passed as an argument to another function.
  • This technique allows a function to call another function.
  • A callback function can run after another function has finished
  • There are two ways in which the callback may be called:
    1. Synchronous

      2. Synchronous callbacks are called immediately after the invocation of the outer function, with no intervening asynchronous tasks.

    2. Asynchronous

      3. While, asynchronous callbacks are called at some point later, after an asynchronous operation has completed.
